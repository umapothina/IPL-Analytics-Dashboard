# IPL-Analytics-Dashboard

An interactive IPL 2025 Analytics Dashboard built using **SQL** and **Power BI** to analyze team performance, player statistics, venue analysis, toss impact, and match outcomes.

---

## 📌 Project Overview

This project transforms IPL 2025 match data into interactive visualizations that provide insights into:

- Team performance
- Player of the Match awards
- Toss win distribution
- Venue analysis
- Highest individual scores
- Head-to-head comparisons

The dashboard allows users to filter data dynamically using slicers for teams, venues, and toss decisions.
